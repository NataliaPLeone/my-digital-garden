A **validação cruzada** é uma técnica  de [[Aprendizado de máquina\|Aprendizado de máquina]] para avaliar a capacidade de generalização de um modelo, a partir de um conjunto de dados.

- amplamente empregada em problemas onde o objetivo da modelagem é a predição. 
- Busca-se então estimar o seu desempenho para um novo conjunto de dados. [^1]

> [!tip] Técnica
> O conceito central das técnicas de validação cruzada é o particionamento do conjunto de dados em subconjuntos mutuamente exclusivos, e posteriormente, o uso de alguns destes subconjuntos para a estimação dos parâmetros do modelo (dados de treinamento), sendo os subconjuntos restantes (dados de validação ou de teste) empregados na validação do modelo.

identifica problemas como: [^2]
- [[Overfitting\|overfitting]]
- [[selection bias\|selection bias]]
